\subsection{Detailed Description}
File Chooser A\+P\+I. 

File Chooser supports A\+P\+Is for file selection dialog. This support attributes of file chooser regarding\+:
\begin{DoxyItemize}
\item allowance of multiple files selection
\item allowance of upload directory
\item list of accepted mime types
\item list of accepted file extensions
\item list of initial selected file names
\item capture attribute for H\+T\+M\+L media capture 
\end{DoxyItemize}
