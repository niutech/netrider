\subsection{Detailed Description}
Web\+Kit frame smart object. 

This object is the low level access to Web\+Kit-\/\+E\+F\+L browser component. It represents both the main and internal frames that H\+T\+M\+L pages contain.

Every ewk\+\_\+view has at least one frame, called \char`\"{}main frame\char`\"{} and retrieved with \hyperlink{ewk__view_8h_af21210605d1aea14d06a507854b10e3b}{ewk\+\_\+view\+\_\+frame\+\_\+main\+\_\+get()}. One can retrieve frame's owner view with \hyperlink{ewk__frame_8h_aca48b5c3aa97f2d25c66c4c395c84271}{ewk\+\_\+frame\+\_\+view\+\_\+get()}. Parent frame can be retrieved with standard smart object's evas\+\_\+object\+\_\+smart\+\_\+parent\+\_\+get(). Children can be accessed with \hyperlink{ewk__frame_8h_ad8158718019258e4403df097b604c426}{ewk\+\_\+frame\+\_\+children\+\_\+iterator\+\_\+new()} or \hyperlink{ewk__frame_8h_a36bda34cf928f11afe7d1db72a40c0ac}{ewk\+\_\+frame\+\_\+child\+\_\+find()}.

Finds a child frame by its name, recursively.

For pre-\/defined names, returns {\itshape o} if {\itshape name} is \char`\"{}\+\_\+self\char`\"{} or \char`\"{}\+\_\+current\char`\"{}, returns {\itshape o's} parent frame if {\itshape name} is \char`\"{}\+\_\+parent\char`\"{}, and returns the main frame if {\itshape name} is \char`\"{}\+\_\+top\char`\"{}. Also returns {\itshape o} if it is the main frame and {\itshape name} is either \char`\"{}\+\_\+parent\char`\"{} or \char`\"{}\+\_\+top\char`\"{}. For other names, this function returns the first frame that matches {\itshape name}. This function searches {\itshape o} and its descendents first, then {\itshape o's} parent and its children moving up the hierarchy until a match is found. If no match is found in {\itshape o's} hierarchy, this function will search for a matching frame in other main frame hierarchies.
