\subsection{Detailed Description}
Describes the context menu A\+P\+I. 

The following signals (see evas\+\_\+object\+\_\+smart\+\_\+callback\+\_\+add()) are emitted\+:


\begin{DoxyItemize}
\item \char`\"{}contextmenu,customize\char`\"{}, Eina\+\_\+\+List $\ast$\+: customize context menu is taken and it gives a list with items of context menu as an argument.
\item \char`\"{}contextmenu,free\char`\"{}, Ewk\+\_\+\+Context\+\_\+\+Menu $\ast$\+: a context menu is freed.
\item \char`\"{}contextmenu,item,appended\char`\"{}, Ewk\+\_\+\+Context\+\_\+\+Menu $\ast$\+: a new item was added to the context menu.
\item \char`\"{}contextmenu,new\char`\"{}, Ewk\+\_\+\+Context\+\_\+\+Menu $\ast$\+: a new context menu was created and it gives the context menu as an argument.
\item \char`\"{}contextmenu,show\char`\"{}, Ewk\+\_\+\+Context\+\_\+\+Menu $\ast$\+: a context menu is shown. 
\end{DoxyItemize}
